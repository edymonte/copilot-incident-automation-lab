# Phase 3 — AI Root Cause Analysis

## How to Use

1. Collect logs from `monitoring/logs/sample-error.log`
2. Collect metrics from `monitoring/metrics/sample-metrics.json`
3. Open Copilot Chat in VS Code
4. Use the prompt from `prompts/root-cause-analysis.md`
5. Paste the logs and metrics as input

## Expected Output

Copilot will produce a structured RCA report:
- Summary
- Evidence
- Impact
- Probable Root Cause
- Proposed Fix
- Risk Level
- Prevention recommendations
